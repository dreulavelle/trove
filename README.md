# Trove — docs site

Source for <https://dreulavelle.github.io/trove/>, built with
[Zensical](https://zensical.org). This is an orphan branch; application code
lives on `main`.

Pushing here triggers `.github/workflows/pages.yml`, which runs
`zensical build` and deploys to GitHub Pages.

Local preview:

```bash
pip install zensical
zensical serve
```
